Pruebas unitarias 

Prueba número 1 : "Creación, modificación, y eliminación de un perro"

 Pasos para el Ingreso y Listado de datos:
  1* Primer paso para poder conectarse al mantenedor de los perris, primero se debe de logearse en la página para que se muestre la opción de "gestión perros", en la cual se puede Agregar un perro con todas sus características ( Foto, nombre , Descripción, Raza y estado) y ademas la opción listar, en donde se muestran todos los perros en la cual se da la opción de modificar y eliminar el perro.

  2* Se  ingresa a la opción "Agregar perro" , en la cual se llenan todos los datos correspondientes de el perro y confirman los cambios.
  
  3* Luego de haber creado un registro, vamos a la opción Listar en la cual se nos muestra un listado con todos los perros con sus respectivas características, en esta vista solo podra editar el administrador puede modificar o eliminar el registro del animal que seleccione.
 Pasos para Modificación o eliminación:
 Modificación:
 
  1* Para poder hacer una modificación o eliminación de un perro previamente registrado en la base de datos, se debe buscar un registro y seleccionar la opcion para editar.
  
  2* Luego de la modificación se busca nuevamente el perro modificado para comprobar de que se realizo la modificación de los datos del animal.
  
  Eliminación:
  1* Para eliminar el registro del perro que queramos debemos ingresar nuevamente al listado de registros para encontrar el animal que se necesita eliminar, una vez encontrado el perro deseado se selecciona la opción y eliminamos nuestro registro de la base de datos.
  
  2* Despues de eliminarlo, nos cercionamos que este no se encuentre en el listado.
 
 Resultados esperados
  Creación, modificación y eliminación de un perro
  
 Resultado final
  Funcionalidad completa
  
 Prueba número 2: "Login de usuario"
 
 Para Posterior ingreso de usuario previamente se debe realizar un registro de usuario valga la rebudancia, el cual se realiza directamente desde la pagina:
 
 Pasos del registro:
 1* Ingresando a la pagina se visualizará la opcion para logearse y registrarse, en este caso nos centraremos en la que dice registro para hacernos nuestro usuario en la base de datos.
 2* Debemos ingresar nombre de usuario, contraseña y correo electronico, una vez ingresado los datos seleccionamos que tipo de usuario esta siendo ingresado.
 
 Ingreso:
 1* Una véz creado el usuario en la base de datos, seguimos el con el paso siguiente que es logearnos en la página, seleccionamos la opción usuarios colocamos nuestros datos y contraseña y ingresamos.
 2* Para salir seleccionamos la opción salir.
 
 Resultados esperados:
 Registro y logeo de usuario en la página "Mis Perris".
 
 Resultado final:
 Registro exitoso y login completo en la página web.
